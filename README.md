# Snakey App By Yadnesh Saindane

A classic snake game web application built with HTML, CSS, and JavaScript, offering a simple and fun way to pass time and test reaction skills.

## Live Demo

Try the live app here: [Snakey App](https://snakey283.vercel.app/)

## Features

- Classic snake gameplay
- Responsive design for all device sizes
- Clean and minimal user interface
- Keyboard controls for smooth snake movement
- Score tracking and display
- Restart option to play again

## Project Structure

- `index.html`: Main HTML file containing the game canvas and interface
- `style.css`: CSS styles for the game visuals and layout
- `app.js`: JavaScript file containing game mechanics, snake movement, food generation, and scoring
- Assets: Images/icons used within the app to enhance UI
